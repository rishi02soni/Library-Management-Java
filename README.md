#  Library Management System (Java)
   
A simple yet effective **Library Management System** built using **Core Java** and **Object-Oriented Programming (OOP)** concepts. This console-based application allows users to manage books within a library and serves as a great be  ginner-friendly project for understanding Java fundamentals. 
                        
---                   
                          
##  Features             
```   
- Add Books to the Library 
- Store Book Information (ID, Title, Author)
- Display Available Books
- Object-Oriented Design
- Easy to Understand and Extend
- Console-Based Application
```
---

##  Class Description

###  Book.java

Represents a book in the library.

**Attributes:**

* Book ID
* Title
* Author

**Methods:**

* Constructor
* Getters
* `toString()`

---

###  Library.java

Manages the collection of books.

**Functions:**

* Add new books
* Display all books
* Store books using `ArrayList`

---

###  Main.java

Application entry point.

**Responsibilities:**

* Create Library Object
* Add Sample Books
* Display Library Records

##  How to Run

### 1️ Clone Repository

```
git clone https://github.com/your-username/library-management-system.git
```

### 2️ Navigate to Project

```
cd library-management-system
```

### 3️ Compile Project

```
javac -d out src/com/example/library/*.java
```

### 4️ Run Application

```
java -cp out com.example.library.Main
```
